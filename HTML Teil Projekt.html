Die Basics zu HTML und CSS

Das hier ist eine einfache Website, die HTML und CSS benutzt. Scroll weiter um mehr zu erfahren! Wir fangen mit den Basics an, es gibt auf jeden Fall wichtige Tags, die ihr für euere Website braucht

Auflistung

  • Doctype html
    • gibt den Dokumenttyp HTML an
  • head
    • beeinhaltet alle Elemente/Informationen, die das Dokument beschreibt
  • title
    • definiert ein Titel für dein Dokument
  • body
    • Sichtbarer Hauptteil deiner Website

In euer Body kommt alles rein, wie zum Beispiel: Bilder
